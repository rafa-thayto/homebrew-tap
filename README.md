# Rafa-thayto Tap

## How do I install these formulae?

`brew install rafa-thayto/tap/<formula>`

Or `brew tap rafa-thayto/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rafa-thayto/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
